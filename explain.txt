1. golang은 package로 이루어져있고 import를 통해서 함수들을 가져옴

2. 함수 맨 앞이 대문자면(ex. Println) 다른 package에서 사용이 가능하고(public)
   함수 맨 앞이 소문자면(ex. mult, repeatMe) 다른 package에서 사용 불가능(private)

3. 변수 선언할 때 맨 앞에 var을 붙여서 변수 const를 붙여서 상수 표시
4. var을 안 쓰고도 (변수명) := (값)을 쓰면 type binding이 compiler가 알아서
5. 파라미터에서 선언 type을 변수 뒤에 붙이고 다 똑같으면 맨 뒤에 하나 붙임(ex. fuunc lenAndUpper )

중요한 점들
1. go는 반환형을 미리 변수로 정할 수 있음
2. return 타입이 여러가지일 수 있음
3. return 타입이 여러가지인 것을 받을 때는 :=를 이용해서 받으면 되고 _를 이용해서
   useless value로 받아서 없애버릴 수 있음(파라미터 몇 개만 필요한 경우)
